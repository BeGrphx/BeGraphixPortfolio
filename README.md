# BeGraphix Portfolio

Portfolio motion design / vidéo IA — **Next.js 15** + **Sanity CMS**.

## Stack

- Next.js 15 (App Router)
- Sanity Studio embarqué sur `/studio`
- Tailwind CSS 4
- Framer Motion (animations scroll & hover)
- Déploiement Vercel

## Démarrage rapide

```bash
npm install
cp .env.local.example .env.local   # puis éditez avec vos identifiants Sanity
npm run dev
```

- Site : [http://localhost:3000](http://localhost:3000)
- CMS : [http://localhost:3000/studio](http://localhost:3000/studio)

Voir **GUIDE.md** pour le pas-à-pas complet (Sanity, local, GitHub, Vercel).
